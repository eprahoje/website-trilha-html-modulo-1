Tecnologia Cliente-Servidor

Voltar

Na arquitetura Client-Server os servidores prestam serviços aos processos dos usuários, isto é: Executam as solicitações de taregas através de um pedido feito pelo cliente e retornam uma resposta, ou seja, devolvem os dados solicitados.

Estrutura Básica do HTML

Voltar

O HTML, que significa Hyper Text Markup Language pode ser criado a partir de uma estrutura básica de tags, que são palavras-chave na página da web que define como o navegador deve formatar e exibir a sua página.

Uma página web pode ser criada com algumas tags básicas, sendo elas:

  • A tag <html> que é o elemento raíz de uma página HTML.
  • A tag <head> que é um elemento que contém meta informações sobre a página HTML.
  • A tag <title> que específica o título de uma página HTML e aparece na barra de título do navegador.
  • A tag <body> que define o corpo do documento, and é contêiner para todos os conteúdos visíveis, como imagens, hyperlinks, tabelas e etc.

Tags de texto

Voltar

As tags de texto no HTML podem ser utilizadas para imprimir texto na página. Aqui vou falar sobre duas tags que são básicas e igualmente importantes para a estrutura da página.

A tag de paragráfro

A tag p define um paragráfro de texto.

As tags de título

As tags h1, h2, h3, h4, h5 e h6 são organizadas de forma que que h1 é a mais importante e h6 a menos importante. É normal que a página tenha apenas uma tag h1 para fins de SEO, é essencial para motores de busca ranquearem melhor a sua página.

Listas ordenadas e não ordenadas

Voltar

Listas não ordenadas são criadas usando a tag ul. Cada item da lista é criado com uma tag li. A característica principal dela é o fator dela ter um simbolo em comum alinhado a esquerda dos itens.

Listas ordenadas são bem similares, porém nelas tem um número antes de cada item.
